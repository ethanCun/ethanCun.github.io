windows服务器常用命令



    1. windows防火墙开放端口：设置  ---  控制面板 ---- windows defender防火墙 ---  高级设置 -- 入站规则 --新建规则 -- 选择需要开放的端口一直next

    2. windows查看防火墙开放的端口列表：Netstat -ano

    3. windows杀死进程： taskkill /f /t /im "进程id或者进程名称"    taskkill /f /pid 4416

    4. windows查看指定端口信息： tasklist |findstr "进程id号"

    5. windows列举所有文件： dir /b

    6. windows进入文件返回上级文件： cd /d 文件夹   上级目录： cd ..
