div水平居中 div垂直居中

让div水平垂直居中的方式

大概效果:

1. 使用弹性布局

下面使用uni-app代码演示：

相当于html：


            <!-- 导航栏开始 -->
            <view class="navi">
                <view class="searchbar">
                    <image class="searchImage" src="../../../static/main/搜索.png"></image>
                    <text class="searchText">商品名称</text>
                </view>
            </view>
            <!-- 导航栏结束 -->

css:


            .navi{

                background: #E12A25;
                height: 128rpx;

                /* 使用flex弹性布局实现 垂直居中水平居中 */
                display: flex;
                align-items: center;/*定义body的元素垂直居中*/
                justify-content: center;/*定义body的里的元素水平居中*/
            }

            .searchbar{

                background: white;
                border-radius: 10rpx;
                width: 60%;
                height: 80rpx;

                /* searchbar的子元素弹性布局 */
                display: flex;
                flex-direction: row;
            }

2. 使用非弹性布局

css：


            ﻿.navi{

                background: #E12A25;
                height: 128rpx;
            }

            .searchbar{

                background: white;
                border-radius: 10rpx;
                width: 60%;
                height: 80rpx;
                display: flex;
                flex-direction: row;

                /* 水平居中 */
                margin: 0 auto;

                /* 垂直居中 */
                /* 默认情况下，由于position的值为static（静止的、不可以移动的）,
                元素在文档流里是从上往下、从左到右紧密的布局的，我们不可以直接通过top、left等属性改变它的偏移。
                所以，想要移动元素的位置，就要把position设置为不是static的其他值，如relative,absolute,fixed等。
                然后，就可以通过top、bottom、right、left等属性使它在文档中发生位置偏移（
                注意，relative是不会使元素脱离文档流的，absolute和fixed则会！也就是说，relative会占据着移动之前的位置，
                但是absolute和fixed就不会 */
                position: relative;
                /* 向下偏移父元素高度的一半 */
                top: 50%;
                /* 再往上偏移子元素高度的一半就垂直居中了 */
                transform: translateY(-50%);
            }
