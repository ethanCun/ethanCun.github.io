html多余的子div靠左显示

1. 父div中多余的子div靠左显示, 并且左右margin刚好平分父div

				
				<style>
					
					html, body{
						
						width: 100%;
						height: 100%;
					}
					
					.par{
						
						width: 100%;
						height: 100%;
						
						display: flex;
						flex-wrap: wrap;
					}
					
					.chd{
						
						/* 
						  关键在于： 计算单个div的宽度和高度， 这样成了响应式的宽高
						  width=100%/n - 2*margin
						 */
						width: calc(25% - 40px);
						height: calc(25% - 40px);
						background: red;
						margin: 20px;
					}
					
				</style>
				<div class="par">
					
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
				</div>

2. 父div中多余的子div靠中显示

				
				<style>
					
					html, body{
						
						width: 100%;
						height: 100%;
					}
					
					.par{
						
						width: 100%;
						height: 100%;
						
						text-align: center;
					}
					
					.chd{
						
						/*
							单个的宽度
							width=100%/n - 2*margin
						 */
						width: calc(25% - 40px);
						height: calc(25% - 40px);
						background: red;
						margin: 20px;
					
						display: inline-block;	
					}
					
				</style>
				<div class="par">
					
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
					<div class="chd"></div>
				</div>
